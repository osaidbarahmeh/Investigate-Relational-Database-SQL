# Investigate a Relational Database

## 📌 Project Overview
This project focuses on exploring and analyzing a relational rental database using SQL and PostgreSQL.

The goal was to extract business insights and improve query performance.

## 🛠 Tools Used
- PostgreSQL
- SQL
- CTE (Common Table Expressions)
- Window Functions
- Aggregations & Grouping

## 🔎 Key Analysis Performed

### 1️⃣ Family Movies Rental Analysis
- Identified family-friendly categories
- Calculated rental frequency per film

### 2️⃣ Store Performance Analysis
- Monthly rental orders per store
- Year-over-year comparison

### 3️⃣ Top 10 Paying Customers
- Monthly payment analysis
- Payment trends during 2007
- Difference between consecutive months using LAG()

## 📊 Example Query Concepts Used
- WITH (CTE)
- DATE_PART()
- DATE_TRUNC()
- LAG() Window Function
- Aggregations (SUM, COUNT)
- GROUP BY & ORDER BY
